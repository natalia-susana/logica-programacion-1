# logica-programacion-1
Ejercicio de comapración de 3 números

## Descripción

Programa desarrollado en **JavaScript** que solicita al usuario el ingreso de 3 números, analiza dichos valores para identificar el número mayor, el número del centro y el número menor, y muestra los resultados ordenados tanto de mayor a menor como de menor a mayor. Adicionalmente, el programa es capaz de detectar si los tres números ingresados son iguales entre sí.

## Tecnologías utilizadas

- **JavaScript**
- **HTML5** (para la ejecución en el DOM)

## Funcionalidades

- Solicita al usuario 3 números mediante `prompt()` y los almacena en sus respectivas variables.
- Analiza los números ingresados para determinar cuál es el **mayor**, cuál es el **del centro** y cuál es el **menor**.
- Imprime los números ordenados de **mayor a menor** y de **menor a mayor**, ya sea por consola o en el DOM.
- Identifica si los tres números son iguales entre sí, mostrando un mensaje indicándolo.

## Ejemplos:

### Caso 1

**Entradas:**
```
4, 4, 2
```

**Salidas esperadas:**
```
4, 4, 2   (orden de mayor a menor)
2, 4, 4   (orden de menor a mayor)
```

### Caso 2

**Entradas:**
```
4, 2, 4
```

**Salidas esperadas:**
```
4, 4, 2   (orden de mayor a menor)
2, 4, 4   (orden de menor a mayor)
```
